Ocean View Resort Reservation System

Ocean View Resort Reservation System is a Java EE web application developed to manage resort operations such as guest registration, room reservations, billing, and reports.

Technologies Used
- Java (JDK 17)
- Java EE (JSP & Servlets)
- Apache Tomcat 9.0.113
- MySQL 8.0
- JDBC
- HTML, CSS, JavaScript
- JUnit (for testing)
- Git (version control)

Features
- Receptionist login system
- Guest registration
- Room reservation management
- Prevent overlapping bookings
- Bill generation
- PDF bill download
- Revenue report generation
- Room and service management
- Admin user management

Default Admin Login
User Name = admin
Password = admin123

How to Run the Project
1. Clone the repository
2. Import the project into IntelliJ or Eclipse
3. Configure MySQL database
4. Run using Apache Tomcat Server
5. Open in browser
